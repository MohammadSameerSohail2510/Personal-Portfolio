# Portfolio Website

A personal portfolio website showcasing my skills, projects, and experience as a Computer Science student specializing in AI & ML.

## Features

- Responsive design
- Smooth scrolling navigation
- Embedded resume viewer
- Project showcase
- Contact information

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Setup

1. Clone the repository
2. Open `index.html` in a web browser
